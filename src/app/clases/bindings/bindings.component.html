Binding

Es una forma de comunicar nuestro template con nuestro component, sin complicarnos tanto como se hace naturalmente en JS

Interpolaci√≥n

Es un mecanismo para poder sustituir una expresion de nuestro TS en nuestro template para poderlo visualizar, para esto nos apoyaremos con dobles llaves {{ '{{}}'}} y dentro de esta ponemos la expresion

Ejemplos:

  1. {{ unMensaje }}
  2. {{ 2+2 }}
  3. {{ retornarSaludo() }}

Property binding (Enlace de propiedades)

Es el enlace que se hace para las propiedades con la sintaxis de [propiedad]="expresion"

Ejemplo:

  2. {{placaOculta}}Exitoso
