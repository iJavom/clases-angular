Http client

El http Client es un modulo que pertenece a angular/common y es el encargado de generar peticiones http a donde nosotros necesitemos.


Los metodos que podemos usar de la libreria son:


  • GET: Sirve para hacer solicitud de datos.
  • POST: Sirve para mandar una peticion con un cuerpo para crear un nuevo elemento.
  • PUT: Sirve para mandar una peticion con un cuerpo para actualizar un elemento.
  • DELETE: Sirve para indicarle a nuestro servidor que queremos borrar un dato.
