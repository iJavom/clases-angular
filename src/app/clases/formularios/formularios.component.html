Formularios


Angular nos ofrece un conjunto de herramientas para manipular campos con el fin de facilitar la comunicacion entre componentes y plantillas.

Template Form (Formularios de plantilla)

Son aquellos formularios que estan constituidos de un conjunto de inputs que poseen nuestro two way binding (enlace bidereccional), es decir el [(ngModel)]. Generalmente se usan para un conjunto de inputs que tienen funcionalidades especificas y no son tan estrictos


para poder hacer uso de estos debemos importar FormsModule

React Form (Formulario Reactivos)


Los formularios reactivos tienen un comportamiento que almacena dentro de un objeto sus inputs y adicional a esto le permite hacer evaluaciones extras con el fin de poder gestionar mucho mejor nuestro formulario. Sabemos que es un formulario reactivo porque forma parte de un formGroup y adicional cada input tiene un formControl


Para nosotros usar formulario reactivos debemos importar ReactiveFormsModule
